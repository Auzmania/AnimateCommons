setup(sym, options)

Setup a flexible Layout

Call this function from Stage to setup a flexible layout with the following options:

  • vertically and/or horizontally center
  • max width and/or height

Parameters

  • AC sym :

    Animate Sybmbol

  • Object options :
    • Library options.lib

      Reference to the library

    • Number options.maxWidth

    • Number options.maxHeight

      Max content height in px. Leave blank or use null for not limitation (default: null)

      Positioning x origin in percent (default: 50)

      Positioning y origin in percent (default: 50)
