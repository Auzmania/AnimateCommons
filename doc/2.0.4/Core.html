(static) EC.CLICK_OR_TOUCH

Constant to equally use click or touch events in event handlers
Can be used in click handler to be optimized on touch devices
The value is eithe 'click' or 'touchstart'

Convert a given symbol to an animated button component that will be played forward and backward on mouse events
A optional "hotspot" element is supported within the symbol to handle all events (mouseenter etc.)

icon String (Optional) path to an icon file (e.g. SVG or PNG) relative to the main HTML file
clickHandler function (Optional) callback for click/touch event
data Object (Optional) data object that will be passed to the button instance and will be available in clickHandler callback as second argument
