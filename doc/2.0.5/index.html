EC.makeStaticButton( sym, [label], [icon], [clickHandle], [data] ) : Boolean

Convert a given symbol to a static button component with three states (normal, over and down)
Implement the three button states (normal, over, down) as labels on the symbol's timeline
A optional "hotspot" element is supported within the symbol to handle all events (click/touch/mouseenter etc.)
Touch enabled: in mobile environments just a tap/click listener will be attached (no states are supported)

ParameterTypeDescription
symAnimateSymbolThe symbol to convert
labelString(Optional) the label of the button (the affected symbol needs to implement a textfield or DIV named "label")
iconString(Optional) path to an icon file (e.g. SVG or PNG) relative to the main HTML file
clickHandlefunctioncallback for click/touch event
dataObject(Optional) data object that will be passed to the button instance and will be available in clickHandler callback as second argument

Returns: (Boolean) always true

Spotlight: Overlay for media (e.g. Images, YouTube) or external Edge Animate compositions

EC.open( config, [documentContext] ) : Boolean

Open a spotlight overlay by passing in a configuration object

ParameterTypeDescription
configObjectThe configuration object 
  var config = {
    width: 700,                                   // Width of the overlay
    height: 400,                                  // Height of the overlay
    borderWidth: 5,                               // Border width (px)
    borderColor: "#FFF",                          // Border color
    type: "image",                                // Media type (image|animate|youtube)
    source: "images/MyImage.png",                 // Source to media
    onClose: function(config, documentContext) {} // Callback for close button
    param: {}                                     // Additional parameter dependent on type (e.g. autoPlay: true for youtube)
  }
documentContextObjectThe parent context for the spotlight overlay (e.g. window.document or window.parent.document)

Returns: (Boolean) returns true whenever the opening of the spotlight overlay was successfull otherwise false

  });
